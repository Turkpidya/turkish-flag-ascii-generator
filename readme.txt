# Enhanced Turkish Flag ASCII Art Generator

This project is a web-based tool that generates ASCII art representations of the Turkish flag. Users can customize the flag's width, ASCII style, and color scheme.

## Features

- Adjustable flag width
- Multiple ASCII styles (Basic, Detailed, Minimal, Dots, ASCII)
- Various color options (Red, White on Red, Black and White, Blue, Green)
- Random style and color options
- Real-time preview
- Copy to clipboard functionality
- Download as TXT file
- Shareable flag configurations via URL

## Usage

1. Open `index.html` in a web browser.
2. Adjust the flag width using the number input (30-100).
3. Select an ASCII style from the dropdown menu.
4. Choose a color scheme from the color dropdown.
5. The flag will update in real-time as you make changes.
6. Use the buttons to copy, download, or share your flag.

## File Structure

- `index.html`: Main HTML file
- `styles.css`: CSS styles for the application
- `script.js`: JavaScript code for flag generation and user interactions
